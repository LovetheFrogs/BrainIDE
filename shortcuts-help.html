Shortcuts

Below you'll find all the shortcuts for BrainIDE. Page is kept up-to-date with new shortcuts.

General shortcuts

Below there's a list of general shortcuts for the main BrainIDE window.

Shortcut Description
Shift+F1 Open BrainIDE help page.
Shift+F2 Open a brainfuck file.
Shift+F3 Create a new file.
Shift+F8 Clear all input fields.
Shift+F9 Execute code.
Shift+F10 Reformat your code in case of a formatting error.
Control+Shift+A Save a file.
Control+Shift+S Save a file as.
Control+Shift+E Close current file.
Control+Shift+O Open a project.
Control+C Copy selection to clipboard.
Control+V Paste clipboard.
Control+X Cut selection to clipboard.
Control+Z Undo changes.
Control+Y Re-do changes.
Shift+ArrowKey Move selection to desired direction.

Translator Shortcuts

This shortcuts are used inside the translator window.

Shortcut Description
Shift+F12 Translate current text to brainfuck.
Control+Shift+C Copy brainfuck code output.
