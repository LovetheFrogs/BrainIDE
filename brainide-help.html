Welcome to BrainIDE! Following some help on how to use this software.

General usage

To use Brain IDE, simply write your code in the big input text area (fig. 1). Once you finish coding, you can move on to executing it. It's as simple as with other IDE's, just hit Run > Run... or use the keyboard shortcut. The output of your code will show under the output dialog box (fig. 2). For programs that take input from the user, type it into the input data box (fig. 3) and run the code as usual.

Advanced help

BrainIDE has several extra functions worth mentioning. Below they will be further explained.

Setting a work directory

The side bar of the IDE shows your project files. That is to say, the .bf files located under your current project directory. If you wanted to change the files that show in here, just go to File > Open Project... and select your desired work directory.

Fix format errors

Although it's not likely formating will fail at any time, if that happens, you could easily fix it by going under Run > Format. This will reformat the whole file and refresh it.
