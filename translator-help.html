Translator

The translator option is one of the most useful features in BrainIDE. What it does is take an string that the user types, and 'translate' it to brainfuck code, helping with one of the most painful things to do in this programing language, outputing text.

How to use it

To use translator, just type the text you want to output using brainfuck, and hit Translate>Translate! (alternatively, you can press Shift+F12). To copy the output, you have to use the Translator>Copy output option or the keyboard shortcut.
